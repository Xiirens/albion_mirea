Функции¶

async req2.load_json(url)[исходный код]¶

Загружает JSON данные по указанному URL.

Параметры:

url – URL для загрузки JSON.

Результат:

Загруженные данные в формате словаря.

async req2.main(urls)[исходный код]¶

Генерирует асинхронные JSON запросы по списку URL.

Параметры:

urls – Список URL для выполнения запросов.

async req2.main_1()[исходный код]¶

Создает словарь и ссылки для запросов к Git и AlbionDataProject.

async req2.start_table(data_sorted, trans)[исходный код]¶

Генерирует таблицы предметов на основе отсортированных данных.

Параметры:

  • data_sorted – Отсортированные данные.

  • trans – Параметры транзакций.

async req2.start(data_sorted, trans)[исходный код]¶

Запускает сортировку и выбор необходимых параметров.

Параметры:

  • data_sorted – Отсортированные данные.

  • trans – Параметры транзакций.

async req2.topografia(data, city_11, city_22, time_update, min_profit, money_profit, trans)[исходный код]¶

Сортирует полученные предметы и генерирует таблицы.

Параметры:

  • data – Данные для сортировки.

  • city_11 – Город 1.

  • city_22 – Город 2.

  • time_update – Время обновления.

  • min_profit – Минимальная прибыль.

  • money_profit – Прибыль в валюте.

  • trans – Параметры транзакций.

async req2.request_to_albion(items)[исходный код]¶

Выполняет дополнительные запросы JSON только по полученным предметам.

Параметры:

items – Список предметов для запросов.

async req2.sort_data(data)[исходный код]¶

Сортирует данные.

Параметры:

data – Данные для сортировки.

async req2.logic_table(name, data, trans, name_img)[исходный код]¶

Создает таблицу данных.

Параметры:

  • name – Имя таблицы.

  • data – Данные для таблицы.

  • trans – Параметры транзакций.

  • name_img – Название изображения.

Параметры:
